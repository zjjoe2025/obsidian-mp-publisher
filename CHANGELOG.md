## [2.2.6] - 2026-03-07

### 🐛 Bug 修复

#### 修复
- **复制/发布样式跨设备不一致**：复制到剪贴板和发布到公众号时，使用 juice 将 CSS 内联到每个元素的 `style` 属性上，确保在任何设备上粘贴或发布后样式都能正确显示
- **列表项多出空行**：发布时无序列表和有序列表项之间出现无效空行，将列表项内部包裹元素从块级 `<section>` 改为行内 `<span>`
- **自定义主题目录路径错误**：本地自定义主题目录使用 `manifest.dir` 获取插件实际安装路径，不再硬编码路径导致在错误位置创建 `custom` 文件夹

## [2.2.5] - 2026-03-06

### 🐛 Bug 修复

#### 修复
- **复制粘贴跨设备样式丢失**：复制到剪贴板时，CSS 依赖 `<style>` 标签而非内联 `style` 属性，导致在不同电脑/浏览器上粘贴到公众号后台时样式不显示。现在复制时通过 juice 将 CSS 内联到每个元素的 `style` 属性上
- **发布到公众号样式丢失**：发布时直接克隆预览 DOM 序列化，`<style>` 标签被公众号后台剥离导致样式全部丢失。改为统一使用 `markdownToHtml()` 生成带内联样式的 HTML

#### 新增
- **纯 CSS 主题系统**：使用 `.mp-content-section` 作为根选择器，通过标准 CSS 文件定义主题样式
- **8 个内置 CSS 主题**：默认、优雅、暗色、极简、清新绿、暖橙、猩红、学术

- **主题管理界面**：通过命令面板「打开主题管理」打开独立的主题管理视图，支持浏览、切换、新建、编辑、删除主题

- **设置页面精简**：设置页面只保留微信公众号 API 配置，主题和样式管理移至独立界面
- **样式内联方式**：发布时使用 [juice](https://github.com/Automattic/juice) 库将 CSS 内联到 HTML，替代旧的手动样式拼接
- **预览视图简化**：移除背景选择器，保留主题选择器、字体选择器和字号调整

#### 移除

## [1.0.0] - 2026-02-05

### 🎉 首次发布

整合 [MP Preview](https://github.com/Yeban8090/mp-preview) 和 [Enhanced Publisher](https://github.com/chararch/obsidian-enhanced-publisher) 两个开源项目，实现**自定义主题 + 一键发布到公众号后台**的完整工作流。

#### 项目背景

每次从 Obsidian 写完内容都要手动复制到公众号后台，然后再调整格式，这个过程经常耗费很长时间。搜索后发现两个符合需求的开源项目：

- **MP Preview**：Markdown 内容转公众号格式预览，可以自定义主题，但不能发布
- **Enhanced Publisher**：支持直接发布 Markdown 内容到公众号后台，但不能自定义主题

本插件将两者结合，实现了**能自定义主题且能一键发布到公众号后台**的完整方案。

#### 功能
- 自定义样式主题预览
- 一键发布到微信公众号草稿箱
- 图片自动上传到微信服务器
- 实时预览公众号效果
- 一键复制格式化内容到剪贴板
- 多种预设主题模板
- 字体和字号调整

#### 致谢
- [MP Preview](https://github.com/Yeban8090/mp-preview) - 样式模板预览系统
- [Enhanced Publisher](https://github.com/chararch/obsidian-enhanced-publisher) - 微信发布功能
